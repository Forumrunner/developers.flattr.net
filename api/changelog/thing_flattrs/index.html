Fetch who flattrd a thing

Now you can get information about who flattred a specific thing. Read more about it in the thing resource.

Example (i have made the response shorter):

Request
GET https://api.flattr.com/rest/v2/things/313733/flattrs
Response
HTTP/1.1 200 OK
Content-Type: application/json;charset=utf-8
X-RateLimit-Limit: 1000
X-RateLimit-Remaining: 999
X-RateLimit-Current: 1
X-RateLimit-Reset: 1342521939
